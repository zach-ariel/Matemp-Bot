# Matemp-Bot
El presente repositorio surge por la necesidad de medir de forma precisa la temperatura a la que se calienta el agua para tomar mate. Se incluye el código para configurar un microcontrolador ESP32 para que mida los valores de temperatura del agua para la bebida y que envíe alertas tanto visuales como de texto via Telegram cuando se alcancen los umbrales de temperatura deseados.

El sistema cuenta con un sensor DS18B20 que tomará los valores de temperatura del agua y tres diodos LED para dar alertas visuales de los estados del dispositivo y el agua. Los resultados obtenidos son enviados al usuario a través de la aplicación de mensajería instantánea Telegram. Una representación del mismo puede encontrarse a continuación.
